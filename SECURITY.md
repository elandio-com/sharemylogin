# Security Policy

## Reporting a Vulnerability

We take the security of **ShareMyLogin** seriously. If you discover a vulnerability, please report it responsibly.

**DO NOT create a public GitHub issue.**

Please create a security report by contacting us at:
**[https://antiscrape.xyz/u/pnqNqe](https://antiscrape.xyz/u/pnqNqe)**

We will acknowledge your report within 48 hours and provide an estimated timeline for resolution.

## Scope

*   **In Scope**: Encryption logic (`src/crypto/`), key derivation, API handling.
*   **Out of Scope**: UI bugs, server infrastructure (unless it affects client security).

## Bounty

We do not currently offer a paid bug bounty program, but we will happily list you in our Hall of Fame if you responsibly disclose a valid security issue.
